# TimeLine
时间线框架



## 配置依赖

在项目的build.gradle中添加
```
allprojects {
    repositories {
        maven { url 'https://www.jitpack.io' }
    }
}
```
在Model的build.gradle中添加 [![](https://jitpack.io/v/ftmtshuashua/TimeLine.svg)](https://jitpack.io/#ftmtshuashua/TimeLine)
```
dependencies {
    implementation 'com.github.ftmtshuashua:TimeLine:version'
}
```
